  • Memory leak: ptr is nullified before freeing

  • NULL Pointer Dereferencing Error: Occurs when a program attempts to access the memory location pointed to by a pointer that has a value of NULL (memory address 0). This usually results in a segmentation fault or program crash.
  • Dangling Pointer: Occurs when a pointer still holds the memory address of a variable that has been deallocated (e.g., memory freed from the heap or a local variable whose scope has ended).
    • Dereferencing a dangling pointer leads to undefined behavior as the memory may now be used for something else.
